# USDX Rewards Platform

## 🔥 Premium Staking & Rewards Platform

Complete production-ready Web3 staking and rewards platform built with Next.js 15, React 19, and TypeScript.

## 📄 Pages Included

- ✅ Landing Page - Hero section with call-to-action
- ✅ Dashboard - Complete stats and activity tracking
- ✅ Staking Page - Membership plans showcase
- ✅ Rewards History - Track all your rewards
- ✅ Referral Program - Earn commissions
- ✅ Leaderboard - Top earners showcase
- ✅ Profile Settings - User account management
- ✅ Admin Dashboard - Platform management

## 🎨 Features

- Premium black & gold theme
- Glassmorphism UI components
- Real charts and analytics (Recharts)
- Responsive design (mobile-first)
- Smooth animations (Framer Motion)
- TypeScript support
- Production-ready code

## 🚀 Getting Started

```bash
npm install
npm run dev
# Open http://localhost:3000
```
